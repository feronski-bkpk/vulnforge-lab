# Что такое VulnForge Lab?

VulnForge Lab — это учебный стенд для практического изучения уязвимостей веб-приложений. 
Стенд содержит несколько уязвимых модулей, каждый из которых демонстрирует определённый тип уязвимости.

## Как запустить стенд

1. Убедитесь, что установлен Docker:
```bash
docker --version
```

2. Добавьте домены в файл hosts:
```bash
echo "127.0.0.1 sqli.vulnforge.local xss.vulnforge.local" | sudo tee -a /etc/hosts
```

3. Запустите стенд:
```bash
cd vulnforge-lab
./vulnforge up
```

4. Откройте в браузере:
- SQL-инъекции: http://sqli.vulnforge.local
- XSS-атаки: http://xss.vulnforge.local

## Как работать с заданиями

Стенд содержит 6 практических заданий разного уровня сложности:

1. Посмотрите список заданий:
```bash
./vulnforge task list
```

2. Начните задание:
```bash
./vulnforge task start 1
```

3. Если застряли — получите подсказку:
```bash
./vulnforge task hint 1       # Первый уровень
./vulnforge task hint 1 2     # Второй уровень
./vulnforge task hint 1 3     # Третий уровень
```

4. Проверьте решение:
```bash
./vulnforge task check 1 "FLAG{...}"
```

## Структура обучения

Каждое задание построено по принципу "от простого к сложному":

1. **Наблюдение** — посмотрите, как работает приложение без атак
2. **Эксперимент** — попробуйте специальные символы, посмотрите на реакцию
3. **Эксплуатация** — выполните атаку и получите флаг
4. **Понимание** — ответьте на вопрос "почему это работает?"

## Доступные модули

### SQL-инъекции (sqli.vulnforge.local)

Демонстрирует три типа SQL-инъекций:
- Обход аутентификации (login.php)
- Извлечение данных через UNION (index.php)
- Слепая инъекция (profile.php)

### XSS-атаки (xss.vulnforge.local)

Демонстрирует два типа межсайтового скриптинга:
- Отражённая XSS (search.php)
- Хранимая XSS (index.php)

## Что дальше?

- Изучите подробные описания уязвимостей в директории `vulnerabilities/`
- Пройдите пошаговые руководства в директории `walkthroughs/`
- Попробуйте найти уязвимости самостоятельно, не заглядывая в подсказки
